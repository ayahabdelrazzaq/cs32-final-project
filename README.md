# cs32-final-project
My CS32 Final Project
original goal: This project is a basketball shooting game inspired by the dramatic style of sports in animes (e.g. Kuroko's Basketball). The player (the user) takes shots at a hoop, and the program computes the movement of the ball and determines whether the shot is successful. The game will include the ability to modify gameplay by acquiring "power ups" or special abilities that increase the probability of the ball going into the hoop.

Im first starting with simulating one basketball shot and determining whether it goes in or misses. This makes the project more manageable while still leaving room to expand it later with additional features like scoring systems and power-ups!
